AUO Smart Expo

2023 AUO Smart Expo official website

This website was designed for 2023 AUO Smart Exposition, and the main purpose of the site was to promote the event and to attract more visitors to register for the expo. The design of this website follows AUO Reshape Design Guideline, which brings an energetic and innovative image to the platform. Being the sole designer, I was responsible for the entire process, including UI and UX design.

Before registering for the event, people usually check the exhibits info and opening hours to help them decide if they want to join. I organized all the material and created a clear information architecture for the platform. Also, CTA buttons are set on each screen, so that visitors can easily find their way to register whenever they want, and as well as a reminder to register. This site not only introduces Featured Exhibits, Schedule and Hours, Direction and Parking and Exhibition Map, but also brings helpful links to the content, helping the audiences find what they need when visiting the website.
