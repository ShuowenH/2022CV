Block page

A responsive block page built with HTML/CSS

Companies block potentially hazardous or risky pages to prevent cyber virus attack. This block page appears when the visited page is suspicious and blocked by Zyxel company intranet. Considering users may visit pages using different devices, the page is designed to be responsive to screen width for better experience. I built this responsive page with HTML/CSS and adorable illustration. Breakpoints are set at 768px and 378px to fit desktop, tablet and mobile.
